UI描述 - 移动、删除

———————————————————

移动和删除需要先选择文件才能点击，否则不能点击，不能跨类型选择，移动/删除只移动/删除当前类型选择的面料

① -当前类型名称

② -移动按钮，选择面料后，点击移动按钮，在面料列表区上方显示半透明遮罩，提示在右侧选择类型进行移动，选择任意类型后，面料移动到该类型中，界面显示的面料列表是移动目标类型下的面料。移动的目标类型可以是新建类型，选择面料后，点击移动，点击添加按钮，添加类型成功后，自动将选择的面料移动到新类型中，并显示新类型列表

③ -删除按钮，选择面料后，点击删除按钮，在删除按钮下方显示确认框提示④，确认则删除，取消则关闭提示框

⑤ -提示：若移动成功则提示移动成功，删除成功则提示删除成功，如有其他情况发生，也在此提示区提示
