UI描述 - 类型列表

———————————————————

① -类型列表：列表按APP内顺序从上到下排列，面料类型标题显示在列表上方，添加按钮置于列表最后

② -类型按钮：

      鼠标滑过或停留在类型按钮时，类型字体颜色变为绿色

      鼠标点击/选中类型时，按钮变为绿底白字

      类型按钮长70px，宽40px，按钮间距10px，圆角5px

      初始默认选择第一个类型，不管什么情况下，始终有一个类型被选中

UI描述 - 添加类型

———————————————————

点击添加按钮，添加按钮下移，下移距离为类型按钮宽度+按钮间距，即40+10=50px

点击后在原添加按钮位置显示输入框，输入框尺寸为170×40，只能输入6个字符

       若用户在类型框中输入文字并回车确认，类型名不重复，则新建该类型，隐藏输入框，在原输入框的位置显示该类型的按钮

       若用户在类型框中输入文字但未回车确认，而执行了其他点击操作，或按键Esc，则取消新建类型，输入框隐藏，且添加按钮回复原位

       若用户未在输入框中输入文字但回车确认了，则按取消新建处理，输入框隐藏，且添加按钮回复原位

       若用户未在输入框中输入文字，而执行了其他点击操作，或按键Esc，则取消新建类型，输入框隐藏，且添加按钮回复原位
